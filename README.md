# Telegram Bot для проверки документов

Бот для проверки PDF документов на ошибки с помощью искусственного интеллекта GigaChat от Сбера. Проверяет морфологические, синтаксические и логические ошибки в текстах.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл .env:

```env
BOT_TOKEN=ваш_токен_бота
DATABASE_URL=postgresql://postgres:password@localhost:5432/tgbot_db
Authorization_Key=ваш_ключ_gigachat
Client_ID=ваш_client_id
Client_Secret=ваш_client_secret
```

3. Создайте базу данных PostgreSQL и запустите бота:
```bash
python bot.py
```

## Как пользоваться

Отправьте `/start` боту и используйте кнопки меню. У каждого пользователя есть 3 бесплатных проверки, после чего нужно выбрать тариф. Администраторы могут создавать новые тарифы через админ-панель.
